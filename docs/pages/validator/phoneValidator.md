# phoneValidator

手机号验证。

中国大陆手机号由11位数字组成，第1-3位为网络识别号；第4-7位为地区编码；第8-11位为用户号码。

## 基本用法

传入一个字符串，如果是手机号格式则返回`true`，否则返回`false`。

```ts
import { phoneValidator } from '@renzp/utils';

phoneValidator('13311111111'); // true
phoneValidator('12000000000'); // false
```

## 参数

| 参数 | 说明         | 类型     | 默认值 | 是否必填 |
| ---- | ------------ | -------- | ------ | -------- |
| v    | 验证的字符串 | `string` | -      | 是       |


## 返回

| 参数 | 说明         | 类型      |
| ---- | ------------ | --------- |
| v    | 是否为手机号 | `boolean` |
