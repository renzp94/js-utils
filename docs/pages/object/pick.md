# pick

获取对象指定属性。

> 注意：此方法不会改变源对象。

## 基本用法

传入一个对象和属性名数组，返回包含指定属性的对象。

```ts
import { pick } from '@renzp/utils'

const a = { a: 1, b: 2 };
const c = pick(a, ["a"]); // c = { a: 1 }  a = { a: 1, b: 2 }
```

## 参数

| 参数   | 说明       | 类型               | 默认值 | 是否必填 |
| ------ | ---------- | ------------------ | ------ | -------- |
| target | 源对象     | `T`                | -      | 是       |
| keys   | 属性名数组 | `Array<K keyof T>` | -      | 是       |


## 返回

| 参数   | 说明               | 类型         |
| ------ | ------------------ | ------------ |
| target | 包含指定属性的对象 | `Pick<T, K>` |
