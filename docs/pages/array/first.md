# first

获取数组第一个元素。

## 基本用法

传入一个数组，返回第一个元素，如果没有则返回默认值。

```ts
import { first } from '@renzp/utils';

first([1, 2, 3]); // 1
first([], 2); // 2
```

## 参数

| 参数         | 说明     | 类型       | 默认值 | 是否必填 |
| ------------ | -------- | ---------- | ------ | -------- |
| list         | 目标数组 | `Array<T>` | -      | 是       |
| defaultValue | 默认值   | `T`        | -      | 否       |

## 返回

| 参数 | 说明               | 类型               |
| ---- | ------------------ | ------------------ |
| v    | 第一个元素或默认值 | `T   \| undefined` |
